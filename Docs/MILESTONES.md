# Milestones

本文件记录课程节点和建议开发阶段。具体玩法目标在团队完成核心设计讨论后再细化。

## Course Dates

- Week 16 Thursday, 2026-06-18: voluntary presentation.
- Week 18 Sunday, 2026-07-02: final submission, including GitHub link and demo video.

## Development Stages

### 1. Project Setup

- 初始化 GitHub 仓库。
- 建立 Unity 项目。
- 确认 Unity 版本。
- 确认 Visible Meta Files 和 Force Text。
- 确认 Git LFS。

### 2. Prototype

- 创建可进入的主场景。
- 实现基础玩家视角和移动。
- 实现最小射击输入。
- 搭建简单测试空间。

### 3. Core Loop

- 明确目标、反馈和胜负条件。
- 实现命中检测。
- 实现计分或任务进度。
- 加入可重复游玩的基础流程。

### 4. VR Interaction

- 接入 Simulator 输入。
- 评估 Quest 真机可能性。
- 如果使用 Quest，补充真机打包和交互规则。

### 5. Polish

- 改善场景视觉。
- 增加音效和 UI 反馈。
- 优化手感和节奏。
- 修复明显 bug。

### 6. Submission

- 录制 demo 视频。
- 检查 GitHub 仓库可访问。
- 整理最终说明。
- 确认提交材料完整。
